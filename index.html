Case Assignment Tool - CRUD

The Case Assignment Tool is a project developed by a data analyst to efficiently assign cases to agents for processing and tracking their progress. The tool is built using a combination of Google Apps Script (GAS), Spreadsheets, BigQuery, JavaScript, HTML, CSS, and Datatable. It incorporates CRUD functionality and offers several key features to enhance case management and agent productivity.

Features

  • Single Case Assignment: The tool allows for the assigning of single cases to agents in "play mode." Each agent is assigned a case, and once the agent closes the case, the next one in the queue is automatically presented. This feature ensures a fair distribution of workload among agents.
  • Queues: Cases are categorized into different queues to provide better organization and tracking. The available queues include:
    • New: Contains newly assigned cases that have not been processed yet.
    • Pending: Contains cases that are awaiting further action or information.
    • OnHold: Contains cases that are temporarily paused or put on hold.
    • Closed: Contains cases that have been successfully resolved or closed.
  • Search and Filter: The tool provides search and filter functionality to quickly locate specific cases based on different criteria such as case number, customer name, or case status. This feature enables efficient case retrieval and management.
  • Re-assignment: In situations where case ownership needs to be changed, the tool allows for re-assigning cases to different agents. This feature ensures flexibility in managing case distribution and workload allocation.
  • Notifications: The tool incorporates a notification system to keep agents informed about important updates and changes in case assignments. Agents receive real-time notifications within the tool interface, ensuring they stay updated and can promptly respond to any changes.
  • Email Integration: To further enhance communication, the tool integrates with email services, enabling automated email notifications to be sent to agents or other stakeholders. This feature facilitates efficient and timely communication regarding case assignments and updates.
  • Admin Dashboard: The tool provides an admin dashboard where managers can track the time spent by each agent on individual cases. This feature allows managers to monitor agent productivity, identify bottlenecks, and make informed decisions for optimizing case assignment and resource allocation.
